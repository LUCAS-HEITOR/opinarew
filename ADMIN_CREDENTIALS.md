✅ **Usuário forte:**
- Contém nome do projeto + Admin + Ano
- Sem espaços, sem caracteres fáceis de adivinhar
- Length: 20 caracteres

✅ **Senha ULTRA segura:**
- 55 caracteres (muito longa!)
- Contém:
  - Letras maiúsculas: L, T, P, S, O, A
  - Letras minúsculas: iv, e, oken, et, al, etc

  - Caracteres especiais: !, @, #, $, &
- Impossível de quebrar por força bruta

## 🔄 Mudar a Senha

Se quiser mudar a senha depois:

1. Abra: `src/context/OpiniaoContext.jsx`
2. Vá para a linha 29-30
3. Mude os valores de `ADMIN_USER` e `ADMIN_PASS`
4. Salve (Ctrl+S)
5. Redeploy no Vercel

## 🆘 Perdeu a Senha?

Se perder a senha, você precisa:
1. Abrir este arquivo
2. Gerar novas credenciais
3. Atualizar o código
4. Fazer redeploy

Por isso **guarde com segurança**! 🔐
